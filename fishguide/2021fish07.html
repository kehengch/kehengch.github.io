# 渔业工程发展历史趋势

海洋牧场：栖息地改造技术、生物资源增殖技术、鱼类行为控制技术（音箱驯化器）、回捕产出管理技术。

  • 萌芽期 放流/基本型
  • 形成期 工程/技术型
  • 转型期 现代/综合型

海洋牧场的建设方式：

  • 人工鱼礁
  • 海底改造+藻礁建设
  • 人工鱼礁+筏式养殖

趋势：

  • 智能化
  • 集成化
  • 虚拟化
  • 数字化与信息化

# 题目

题目

  1. 【判断题】 海洋牧场建设包括4项核心技术内容：栖息地改造技术，生物资源增值技术，鱼类行为控制技术，回捕产出管理技术。 对

  2. 增值放流就是用人工方式向海洋、江河、湖泊等公共水域放流水生生物苗种或亲体的活动。 对

  3. 人工鱼礁材料主要有下面哪几种组成： A、钢筋混凝土 B、塑料 C、钢质材料D、其它废弃物

  4. 我国人工鱼礁建设在上世纪80年代有所搁置，上世纪90年代至2000年又再次被发展，现在一些地方已经初具规模。 对

  5. 渔业生态工程未来主要有哪几种趋势？ A、信息化和数字化B、虚拟化C、智能化D、集成化

  6. 我国网箱养殖发源于淡水，而后逐渐向海洋延伸。 对
