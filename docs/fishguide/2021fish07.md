## 渔业工程发展历史趋势

**海洋牧场**：栖息地改造技术、生物资源增殖技术、鱼类行为控制技术（_音箱驯化器_）、回捕产出管理技术。

- 萌芽期
放流/基本型
- 形成期
工程/技术型
- 转型期
现代/综合型

海洋牧场的建设方式：

- 人工鱼礁
- 海底改造+藻礁建设
- 人工鱼礁+筏式养殖

趋势：

+ 智能化
+ 集成化
+ 虚拟化
+ 数字化与信息化
